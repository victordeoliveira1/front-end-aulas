# Repositório de Estudos Front-End

Este repositório foi criado para armazenar conteúdos, exercícios e projetos desenvolvidos durante as aulas de **Front-End**.

O objetivo é registrar a evolução nos estudos, praticar conceitos fundamentais do desenvolvimento web e servir como base para consultas futuras.

---

## 📌 Sobre o Repositório

Aqui estão organizados materiais utilizados nas aulas, exemplos práticos e exercícios relacionados ao desenvolvimento de interfaces web.

Conteúdo voltado para aprendizado de:

- Estruturação de páginas web
- Estilização com CSS
- Lógica com JavaScript
- Componentização
- Responsividade
- Boas práticas de desenvolvimento front-end

---

## 🚀 Tecnologias Utilizadas

- HTML5
- CSS3
- JavaScript
